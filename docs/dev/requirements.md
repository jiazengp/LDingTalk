# 依赖

## 安装依赖

### 如果你使用系统解释器

```shell
pip install -r requirements.txt
```

### 如果你使用 Virtualenv 环境

```shell
venv\Scripts\python.exe -m pip install -r requirements.txt
```

## 导出依赖

### 如果你使用系统解释器

```shell
pip freeze > requirements.txt
```

### 如果你使用 Virtualenv 环境

```shell
venv\Scripts\python.exe -m pip freeze > requirements.txt
```
